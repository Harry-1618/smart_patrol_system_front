# 智能巡更系统

基于摄像头的智能巡更系统，用于管理小区物业巡更人员的巡更工作。

## 功能特点

- 实时监控巡更人员轨迹
- 自动识别巡更人员身份
- 巡更数据统计和报表生成
- 缺勤数据上报和审批
- 摄像头故障管理
- 多角色权限管理

## 系统角色

1. 系统管理员
   - 用户管理
   - 系统配置
   - 权限管理

2. 物业主管
   - 巡更数据审核
   - 缺勤审批
   - 故障处理
   - 报表查看

3. 巡更人员
   - 巡更任务执行
   - 个人巡更记录查看
   - 缺勤申请

## 技术栈

- 前端框架：Vue 3
- UI 组件库：Element Plus
- 状态管理：Pinia
- 路由管理：Vue Router
- 构建工具：Vite

## 开发环境要求

- Node.js >= 16.0.0
- npm >= 7.0.0

## 安装和运行

1. 安装依赖
```bash
npm install
```

2. 启动开发服务器
```bash
npm run dev
```

3. 构建生产版本
```bash
npm run build
```

## 项目结构

src/
